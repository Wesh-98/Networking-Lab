### General Tips:
- Use clear, high-resolution screenshots
- Name files descriptively
- Include date in filename if tracking progress
- Consider using a screenshot tool with annotation capabilities

## How to Add Screenshots:

1. Take screenshot on your computer
2. Name it appropriately
3. Copy to this folder
4. Reference in your documentation
5. Commit to git

Example in markdown:
```markdown
![VLAN Configuration](../screenshots/project1-vlan-brief.png)
```
