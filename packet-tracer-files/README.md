# Packet Tracer Files

Save your Packet Tracer (.pkt) files here.

## Project Files:

### Recommended Structure:
- `project1-vlans.pkt` - Project 1 complete topology
- `project2-dhcp.pkt` - Project 2 with DHCP
- `project3-nat.pkt` - Project 3 with NAT/PAT
- `project4-acls.pkt` - Project 4 with ACLs
- `project5-security.pkt` - Project 5 with security features
- `project6-troubleshooting.pkt` - Project 6 troubleshooting scenarios

### Incremental Saves:
Consider saving at each major milestone:
- `project1-step1-vlans-created.pkt`
- `project1-step2-trunk-configured.pkt`
- `project1-step3-router-configured.pkt`
- `project1-complete.pkt`

## How to Save Packet Tracer Files:

1. In Packet Tracer: File → Save As
2. Navigate to this folder
3. Name descriptively
4. Click Save
5. Commit to git

## Tips:

- Include descriptions in the PT file (File → File Info)
- Test your configuration before saving
- Keep backups of working configurations
- Document any special configurations in the file

## Opening Files:

1. Download repository
2. Navigate to `packet-tracer-files/`
3. Double-click .pkt file
4. Opens in Packet Tracer

---

*Note: .pkt files are binary and may be large. Consider using Git LFS for files over 50MB.*
